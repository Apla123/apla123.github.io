Язык программирования контрактов Simvolio

Simvolio - это язык написания смартконтрактов с быстрой компиляцией в байткод. Simvolio поддерживает стандартный набор операторов и конструкций, функции для работы с таблицами базы данных и систему обработки ошибок. Код контрактов пишется в редакторе программного Documentation клиента Molis.

База данных экосистемы

Каждая экосистема имеет собственную базу данных. Инструменты для управления базой данных обеспечивают создание таблиц, добавление колонок и назначение прав доступа.

Собственный язык шаблонизатора Protypo

Пользовательские интерфейсы приложений пишутся на специальном языке шаблонизатора Protypo. Функции и конструкции языка Protypo обеспечивают получение значений из базы данных, построение форм для передачи значений в контракты и выполнение других операций.

Код страниц интерфейсов создается в специальном редакторе программного клиента Molis.

Визуальный конструктор интерфейсов

Визуальный конструктор интерфейсов позволяет легко и быстро настроить внешний вид страниц приложений без обращения к HTML коду. С его помощью на страницу добавляются текстовые блоки, элементы форм и навигации, таблицы и диаграммы, а так же настраиваются их стили, положение, размеры.

Механизм контроля прав доступа

Одним из главных преимуществ Aplachain по сравнению с другими блокчейнплатформами является наличие в программной среде экосистем многоуровнего механизма управления правами доступа.

Права доступа устанавливаются на все операции по созданию и изменению контрактов, таблиц базы данных, интерфейсов.

Права доступа могут предоставляться членам экосистемы, ролям и смартконтрактам.
